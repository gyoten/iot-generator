# Iot Generator

Iot Generator make it easy to set development environment up for mbed HRM1017.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'iot'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install iot
